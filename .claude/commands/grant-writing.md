# 國際計畫撰寫框架（grant-writing）

> ⚠️ 本 Skill 為 **NSF/NIH 等美國機構**的計畫撰寫框架。
> 國科會（台灣）計畫請使用 `nstc-grant.md`，勿混用。
>
> 本 Skill 為 `claude-scientific-skills/scientific-skills/research-grants/SKILL.md` 的本地封裝。

## 使用 Agent

- **主要**: `grant-writer`（僅作結構參考，不直接產出）

## 使用時機

- 參考 NSF 的 Intellectual Merit + Broader Impacts 結構邏輯
- 參考 Modular Budget 預算結構設計思維
- **實際產出國科會計畫書一律使用 `nstc-grant` Skill**

## NSF vs 國科會差異（對照表）

| 面向 | NSF (本 Skill) | 國科會 (nstc-grant) |
|------|----------------|---------------------|
| 核心章節 | Intellectual Merit + Broader Impacts | 研究目的 + 文獻探討 + 研究方法 + 預期成果 |
| 預算 | Modular Budget | 人事費/業務費/設備費 |
| 語言 | 英文 | 中文為主（摘要含英文）|
| 頁數 | 15 頁 | 10-25 頁（依計畫類型）|
